[Atividade] Tags de Código

code:

A tag code é utilizada para destacar uma parte de código que será também exibidada no navegador.

Exemplo: link rel="stylesheet" href="tags.css">

samp:

A tag samp faz com que o texto em que é marcado, apareça no navegador com a fonte monoespace, ou qualquer outra fonte que o programador escolher.

Exemplo: HTTP 404 - Infelizmente não foi possível encontrar a página que você solicitou

pre:

A Tag 

pre

faz com que quando usada o texto marcado por ela fique igual o escrito no corpo html.

Exemplo: 

 
    font-family: Arial, Helvetica, sans-serif;
    text-align: justify;

kbd e atalhos mais usados:

A tag kbd que é utilizada para orientar usuários em instruções normalmente "on line", ou impressas, como atalhos.

Exemplos:

  • Ctrl + Shift + S - Salva todos os arquivos abertos.

  • Ctrl+ Z - Desfaz a última ação (uso muito)

  • Alt+ Z - Para quebrar as linhas

  • Ctrl+ Shift + P - Abre a caixa de comandos, onde é possível pesquisar por funcionalidades específicas.
